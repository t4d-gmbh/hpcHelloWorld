# Exo 0: Introduction

- Create your own repository by "Use this template"
- If you want to have exemplary solutions for the exercises included, be sure to select "Include all branches" when creating a new repository based on this template.
- Once you've created your own copy of the repository, you're ready to go.
- Head over to [Exercise 1](./Exo_1.md) to get started.
- If you have questions you might check the exemplary solutions available under the corresponding branches, i.e. solutions to _Exercise 1_ can be found in branch _exercise_1_.
  Alternatively, you can also [open an issue](https://github.com/t4d-gmbh/hpcHelloWorld/issues/new) on the template repository or checkout the [discussions](https://github.com/t4d-gmbh/hpcHelloWorld/discussions).
